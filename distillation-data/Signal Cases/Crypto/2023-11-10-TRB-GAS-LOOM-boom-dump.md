# TRB/GAS/LOOM 数倍爆拉案例

## 案例概述

川沐分析了多个山寨币爆拉数倍后的共同特征，形成了系统性的风险识别框架。

## 共同信号特征

1. **期现巨大差价**：现货与合约价格分离 30-50%
2. **负费率拉高**：从 0% 拉至 1.5-2.5%
3. **临死反抽**：庄家砸盘前最后一次拉升诱多
4. **持仓量变化**：爆拉前同步增长，结束时骤降

## 操盘流程

```拉高现货 → 制造期现基差 → 诱散户进入期货 → 提供砸盘流动性 → 平多建空 → 砸盘
```

## 引用原文

> "庄家体量巨大，必须制造流动性才能安全出货。期现价差是核心信号。"
