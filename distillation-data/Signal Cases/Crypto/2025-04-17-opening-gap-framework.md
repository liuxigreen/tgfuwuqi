# 开口空洞框架（2025年迭代）

## 框架定义

山寨币爆拉结束时，名义价值继续涨但持仓量骤然分离，这是庄家在平多建空。

## 关键观察指标

| 指标 | 状态 | 含义 |
|------|------|------|
| 名义价值 | 继续上涨 | 市场表面仍在爆拉 |
| 合约持仓量 | 骤然下降 | 庄家平多转空 |
| 合约持仓量 | 骤然上升 | 庄家在接盘散户多单 |

## 庄家操盘流程

```爆拉前持仓同步增长 → 爆拉结束时出现开口空洞 → 庄家平多建空 → 横盘/小拉诱多 → 砸盘
```

## 与早期框架的区别

| 版本 | 时间 | 核心信号 | 复杂度 |
|------|------|----------|--------|
| V1 | 2023-09 | 负费率+期现差价 | 双信号 |
| V2 | 2023-11 | 持仓量+期现差价+费率 | 三信号 |
| **V3** | **2025-04** | **开口空洞（名义价值 vs 持仓量）** | **单一可视化信号** |
