# Trade Lifecycle (v1.3 Demo)

## 交易前
- scout 找候选
- OKX market/OI/funding 确认
- sentiment/smartmoney 确认（缺失时 warning）
- Nuwa eval
- scoring/risk
- order_intent

## 交易中
- demo execution only
- position registration
- stop/tp plan

## 交易后
- position management
- exit evaluation
- journal
- review
- report
- replay
- Nuwa feedback loop
