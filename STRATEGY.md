# AgentHansa 完整策略

## 平台规则

- **反作弊**: 需要真实证明 URL、指定内容、AI 评分 A-F
- **Spam 惩罚**: spam = $0 + 升级禁赛 (20min → 永久)
- **API 提交间隔**: 30秒+，避免 spam ban

---

## LLM 资源配置

### 免费模型 (claude-sonnet-4-5 + claude-haiku-4-5)
- **Endpoint**: https://newapi.lzgzxs.xyz/v1
- **用途**: 日常判断、分类、简单推理、前置条件检查
- **Key**: sk-HEhOnrseTT... (Sonnet) / sk-HJi3Q8KWVp... (Haiku)

### GPT-5.4 (bankofai.io 轮换)
- **Endpoint**: https://api.bankofai.io/v1
- **用途**: 复杂推理、数学题解答、最终决策
- **Key**: 13个，每个 ~$1 额度，80次/个 安全阈值
- **预计总量**: 13×80 ≈ 1040 次调用

## 双模型工作流：写 → 审 → 提交

```
收到红包任务
    │
    ▼
[0] 前置检查 (Haiku-4-5) ───── 免费
    ├─ 链接是否可用？
    ├─ 证明 URL 是否有效？
    ├─ 平台是否在线？
    └─ 如果任一失败 → 发 TG 通知，跳过
    │
    ▼
[1] 写答案 (Writer) ───────── Sonnet-4-5 (免费)
    ├─ 读取题目/任务内容
    ├─ 生成完整答案
    └─ 输出结构化结果
    │
    ▼
[2] 审核修正 (Reviewer) ───── GPT-5.4 (轮换 key)
    ├─ 检查答案正确性
    ├─ 修正计算/逻辑错误
    ├─ 确保符合证明格式要求
    └─ 输出最终版本
    │
    ▼
[3] 提交
    ├─ 等待 30s+
    ├─ 提交最终答案
    ├─ 检查返回状态
    └─ 如果 paused/banned → 立即停止提交，发 TG 通知
```

### 异常路径
```
审核阶段发现题目需要极强推理？
    → 直接升级为 GPT-5.4 重写 + 自审
审核阶段发现题目不可完成？
    → 发 TG 通知，跳过
```

---

## 错误处理

| 情况 | 操作 | 通知 |
|------|------|------|
| 证明 URL 无效 | 跳过 | TG |
| 链接不可用 | 跳过 | TG |
| 数学题解不出 | 跳过 | TG |
| 被 ban/paused | 停止提交 | TG |
| GPT-5.4 key 用尽 | 自动轮换 | 无 |
| 所有 key 用尽 | 降级到免费模型 | TG |
| 提交成功 | 记录 | 无 |

## 模型分工

| 角色 | 模型 | 费用 | 用途 |
|------|------|------|------|
| **前置检查** | Haiku-4-5 | 免费 | 链接/URL/平台可用性验证 |
| **Writer** | Sonnet-4-5 | 免费 | 生成答案初稿 |
| **Reviewer** | GPT-5.4 | 轮换 key | 审核、修正、确保正确性 |

### 消耗估算
- 每个红包 = 1次 Sonnet(免费) + 1次 GPT-5.4(轮换)
- 13 个 key × 80 次/个 ≈ **1040 个红包**（GPT-5.4 侧）
- Sonnet-4-5 无限（免费）
